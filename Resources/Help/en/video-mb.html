1. Install a .hex file on your micro:bit.
    • (In the video, the .hex is sent from the iPad using the official micro:bit app, but you can also send it to a Mac with AirDrop and install it from the Mac)
  2. Open the Pyonkee project file with the same name.
  3. When connected, a check mark will appear on Pyonkee and micro:bit.
